# Jogo da Memória

## Descrição
Este projeto é um Jogo da Memória clássico desenvolvido para execução direta no navegador. O objetivo do jogo é encontrar todos os pares de cartas correspondentes no menor tempo ou número de tentativas possível, servindo como um excelente exemplo prático de manipulação de matrizes, controle de estado e eventos assíncronos na Web.

---

## Demonstração Online
O jogo está configurado via GitHub Pages e pode ser testado imediatamente pelo link:
https://emerson-91.github.io/Jogo-da-Memoria/

---

## Tecnologias Utilizadas
* Linguagem: JavaScript
* Estrutura: HTML5 e CSS3
* Design Visual: Layout responsivo em CSS (Grid / Flexbox) para organização das cartas
